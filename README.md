# Vianova_website
Vianova is a modern niche perfume e-commerce website developed as a Web Technologies course project by Software Engineering students at Kocaeli University.

## About the Project

Vianova aims to provide users with a premium online shopping experience for niche perfumes. The website features a luxury-inspired design, perfume collections, product detail pages, campaigns, favorites system, and interactive user experience elements.

## Features

* Modern and responsive user interface
* Luxury-themed homepage design
* Women's and Men's perfume collections
* Product listing and filtering
* Product detail pages
* Favorites page
* Campaigns and special offers section
* Customer reviews section
* Perfume recommendation quiz
* Collection showcase page
* Animated hero sections and visual effects

## Technologies Used

* HTML5
* CSS3
* JavaScript
